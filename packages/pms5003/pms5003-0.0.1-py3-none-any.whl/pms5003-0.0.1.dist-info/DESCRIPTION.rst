Installing
==========

Stable library from PyPi:

-  Just run ``sudo pip install pms5003``

Latest/development library from GitHub:

-  ``git clone https://github.com/pimoroni/pms5003-python``
-  ``cd pms5003-python``
-  ``sudo ./install.sh``

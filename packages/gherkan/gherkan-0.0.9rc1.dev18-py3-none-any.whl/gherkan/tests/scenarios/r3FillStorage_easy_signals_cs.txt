# language: cs

Požadavek: robot R3
    Robot R3 má na starosti skládání Kabiny. Robot postupně sbírá kostičky potřebené ke kompletaci.
Kontext:
  Pokud lineOn

Scénář: Pokud je sklad prázdný, když je vozíček XX ve stanici XY, pak robot začne vykládat všechny kostičky 2x2.
  Pokud storageCount == 0
  Když shuttleXXAtStationXY
  Pak robotr3ProgramNumber == 1 && robotr3ProgramStarted

Scénář: Když robot skončil vykládání kostiček 2x2, pak robot R3 začne vykládat všechny kostičky 2x4.
  Když robotr3ProgramNumber == 1 && robotr3ProgramEnded
  Pak robotR3ProgramNumber == 2 && robotR3ProgramStarted
